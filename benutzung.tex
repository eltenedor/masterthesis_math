\chapter*{Benutzung der Vorlage}
\label{chapter:benutzung}

Die Vorlage sollte wie folgt benutzt werden:

\begin{enumerate}[1.]\item Der gesamte Inhalt von \verb|VorlageNichtTUDDesign.rar| sollte in einen Ordner extrahiert werden.

		\item In der Datei \verb|thesisnichttuddesign.tex| werden alle notwendigen Dateien eingebunden und das Format festgelegt. 
		
		Sie können dabei in \verb|pakete.tex| und \verb|befehle.tex| Änderungen vornehmen.
		
		\item Alle inhaltlichen Kapitel werden separat per \verb|\include{}| eingebunden und für jedes Kapitel erstellen Sie eine Datei \verb|kapitel1.tex| usw.
		
		\item In \verb|titel.tex| ändern Sie an den entsprechenden Stellen die Titelseite.
		
		\item In \verb|einleitung.tex| verfassen Sie die Einleitung inklusive Danksagung.
		
		\item Im Anschluss binden Sie Ihre Kapitel ein. In dieser Vorlage sind es die Kapiteldateien \verb|benutzung.tex| und \verb|hinweise.tex|.
\end{enumerate}
